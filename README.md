## Installation

```bash
$ npm install
```

## Seeding scripts

```bash
# push concearned tables to db (calls a rest before hand)
$ npm run seed:run

# clear and reset concearned tables in db
$ npm run seed:reset
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
